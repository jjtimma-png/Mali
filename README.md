# MaLi Unique Boutique (Start Over Build)

Fresh React + Vite + Tailwind storefront with SEO blog, categories, and checkout (demo).
Includes `netlify.toml` and `vercel.json` for smooth deploys.

## Quick start
npm install
npm run dev

## Build
npm run build
npm run preview

## Netlify
Build: npm run build
Publish: dist
SPA fallback handled by netlify.toml

## Vercel
Framework: Vite
Build: npm run build
Output: dist
