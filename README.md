# MaLi Unique Boutique

A React + Vite + Tailwind boutique storefront with cart, checkout (demo), and SEO blog.

## Brand
- **Name:** MaLi Unique Boutique
- **Slogan:** Experience elegance feeling confidence in your style when shopping at MaLi Unique boutique
- **Colors:** Light red & white (primary: `#ff4d5a`)

## Run Locally
```bash
npm install
npm run dev
```
Open the URL shown by Vite (usually http://localhost:5173).

## Build
```bash
npm run build
npm run preview
```

## Deploy (quick)
- **Vercel**: Import this folder -> Framework: Vite.
- **Netlify**: Build command `npm run build`, Publish directory `dist`.
